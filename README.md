# Incipit

A lightweight desktop LaTeX editor built with Tauri, React, and TypeScript that provides real-time PDF compilation using Tectonic, featuring a CodeMirror-based editor with syntax highlighting, project management with file tree navigation, split-panel layout for side-by-side editing and preview, and support for multi-file LaTeX projects with automatic dependency resolution.
